# Getting Started with weather React App
# COMP 3123 lab test 2 - weather app 
# student id 101485443


This project is simple react based weather application that allows users to search for real time weather information for any city. the app retrieves data from the open weather app and displays key details such as temprature, humidity, wind.

## features 
- search by city names
- displays: 
 - city and country
 - temprature in C
 - weather condition and desription
 - weather icons from open weather app
 - wind speed and humidity
- uses modern UI with grdaeint background and clean layout
- Error handling (nothing to geocode, city not name )
-Hosted on vercel

- **open weather map api**
